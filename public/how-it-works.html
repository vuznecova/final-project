How it works

In this application, exposure therapy is used to help overcome the fear of heights. The user starts with simple 360° panoramas and gradually progresses to more challenging levels of exposure.

Main Steps

Select a Level

The user chooses an initial exposure level (e.g., low heights) to adapt and get accustomed to the virtual environment.

VR Immersion

Realistic height sensations are created using 3D panoramas or 360° videos, but in a safe environment.

Anxiety Tracking

The user rates their anxiety level at each stage, gradually acclimating and noticing a reduction in fear.
